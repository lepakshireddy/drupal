INTRODUCTION
============
This module interact with salesforce CRM(Veeva) using JBoss webservices API's to do the following functionality.

 * Event list
 * Event detail
 * Rsvp events
 * HCP events
 * Event survey
 * Event settings
 * event mail
 * Calender email

REQUIREMENTS
============
This module requires the following modules

 * jjfusion_ap_hcp_profile module.
 * CTools (https://www.drupal.org/project/ctools)
 
Portal Name(Domain) should be create in CRM portal table before install/enable the module.
 
INSTALLATION
============
Copy jjfusion_ap_hcp_events folder  to your module directory and then enable on the admin modules page.
