INTRODUCTION
============
This module interact with salesforce CRM(Veeva) using JBoss webservices API's to do the following functionality.

 * Create User Profile
 * User Login
 * Forgot Password
 * Reset Password
 * Picklist Values 

REQUIREMENTS
============
This module requires the following modules

 * Clientside Validation (https://www.drupal.org/project/clientside_validation)
 * FAPI Validation (https://www.drupal.org/project/fapi_validation)
 * CTools (https://www.drupal.org/project/ctools)
 
Portal Name(Domain) should be create in CRM portal table before install/enable the module.
 
INSTALLATION
============
Copy jjfusion_ap_hcp_profile.module to your module directory and then enable on the admin modules page.
